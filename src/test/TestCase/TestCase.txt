Тест-кейс ID 2
name - Проверка наличия на странице "Компьютеры и офис" фильтра "Бесплатная доставка"
Tag - Critical
Шаги
1. Зайти на главню страницу https://aliexpress.ru
2. В меню слева убедиться, что есть меню "Категории"
3. Нажать на меню "Категории"
4. Нажать на иконку "Компьютеры и офис"
Ожидаемый результат - открылась страница "Компьютеры и офис", в блоке слева есть возможность выбора "Бесплатная доставка"

Тест-кейс ID 3
name - Проверка поиска товара в поисковой строке
Tag - Blocker
Шаги
1. Зайти на главню страницу https://aliexpress.ru
2. В поисковой строке набрать Pixel 7
3. Нажать Enter
Ожидаемый результат - в результате запроса появятся товары Google Pixel 7

Тест-кейс ID 4
Tag - Blocker
name - Проверка возможности добавления найденного товара в Корзину, проверка работоспособности кнопки "Заказать у этого продавца"
Шаги
1. Зайти на главню страницу https://aliexpress.ru
2. В поисковой строке набрать Pixel 7
3. Нажать Enter
4. Выбрать первый товар в поисковой таблице нажатием на него левой кнопки мыши
5. В открытой в новом окне вкладке нажать кнопку "Добавить в корзину"
6. Нажать на кнопку "Перейти в корзину"
7. В открывшемся окне убедиться, что в Корзине есть данный товар, нажать на кнопку "Заказать у этого продавца"
Ожидаемый результат у незарегистриванного пользователя - должно выскочить окно с Авторизацией/Регистррацией
